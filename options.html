Copy Title & URL

Extension Options

code Available Variables

  • <title> - The page title
  • <url> - The page URL
  • <quote> - Selected text content

help_outline How it Works

Action Types:

  • Copy to Clipboard: Default action.
  • Open URL: Activates if format starts with a URL scheme (e.g., `https://`).

Keyboard Shortcut:

Activate with Cmd+Shift+1. Customize at chrome://extensions/shortcuts.

lightbulb Examples

Tip: Click any example to copy it.

  • Markdown:
    [<title>](<url>)
  • Quote with source:
    "<quote>"\n\nSource: <title> - <url>
  • Search with ChatGPT:
    https://chatgpt.com/hint-search?q=Summarize%20<url>
  • Search selected text:
    https://google.com/search?q=<quote>
  • Launch macOS Shortcuts:
    shortcuts://run-shortcut?name=TestInput-foo
