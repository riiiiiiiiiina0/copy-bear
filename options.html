Keyboard Shortcut:

The default keyboard shortcut to activate the extension is +Shift+1. Activating the extension with the shortcut is the same as clicking the extension icon. You can press the shortcut multiple times quickly to simulate double-clicking or triple-clicking the icon.

You can customize this shortcut by visiting chrome://extensions/shortcuts.
